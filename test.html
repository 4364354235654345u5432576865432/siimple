# Forms

Input text

Add a class .form-input to your text-based input fields. 
<input type="text" class="form-input" placeholder="Put your text here">

Additionally, you can add a disabled attribute to your input text field:

<input type="text" class="form-input" placeholder="Disabled input" disabled>

Input submit

You can add the class .form-input to your submit input fields: 
<input type="submit" class="form-input" value="Submit form">

Select

Add the class .form-select yo your drop-down list tag: 
<select class="form-select">
 <option>Element 1</option>
 <option>Element 2</option>
 <option>Element 3</option>
 <option>Element 4</option>
</select>

Textarea

Simply add the class .form-textarea and modify the attribute rows with the number of rows. 
<textarea class="form-textarea" rows="4" placeholder="Your text"></textarea>

# Buttons

Default button

Add a .btn class to transform your div, a, etc... into a button. Optionally, add a .btn-[color] class for modifying the color.
Default button
Green button
Orange button
Red button
Purple button
<div class="btn">Default button</div>
<div class="btn btn-green">Green button</div>
<div class="btn btn-orange">Orange button</div>
<div class="btn btn-red">Red button</div>
<div class="btn btn-purple">Purple button</div>

Additionally, you can change the button size adding a .btn-small or .btn-big class.

Small button
Big button
<div class="btn btn-blue btn-small">Small button</div>
<div class="btn btn-green btn-big">Big button</div>

Outline buttons

If you wan an outline button, add the .btn-outline class to your element:
Outline button
<div class="btn-outline">Outline button</div>

# Alerts

 You can create a blue alert adding the .alert class to your div.
This is an alert, and this is a link inside the alert.
<div class="alert alert-blue">This is an alert.</div>

You can change the alert color adding .alert-green .alert-orange and .alert-red for make the alert green, orange and red respectively.

This is a green alert, and this is a link inside the alert.
This is an orange alert, and this is a link inside the alert.
This is a red alert, and this is a link inside the alert.
<div class="alert alert-green">This is a <b>green</b> alert.</div>
<div class="alert alert-orange">This is a <b>orange</b> alert.</div>
<div class="alert alert-red">This is a <b>red</b> alert.</div>

Also, you can add an icon to your alert. These are the available icons:

This is a success alert.
This is a warning alert.
This is an error alert.
<div class="alert alert-success">This is a <b>success</b> alert.</div>
<div class="alert alert-warning">This is a <b>warning</b> alert.</div>
<div class="alert alert-error">This is an <b>error</b> alert.</div>

# Panels

 A panel consists in one div with the class .panel, and two div inside, the first with the class .panel-head and the second with the class .panel-body.
Panel
This is the content of the panel
<div class="panel">
<div class="panel-head">Panel</div>
<div class="panel-body">This is the content of the panel</div>
</div>

# Tables

Basic table

Simply add .table to the class of your table.
# Col1 Col2 Col3
Row1 Item Item Item
Row2 Item Item Item 
<table class="table"> ... </table>

Hover table

Add .table-hover to the class of your table.
# Col1 Col2 Col3
Row1 Item Item Item
Row2 Item Item Item 
<table class="table table-hover"> ... </table>

Striped rows table

Add .table-striped to the class of your table.
# Col1 Col2 Col3
Row1 Item Item Item
Row2 Item Item Item
Row3 Item Item Item
Row4 Item Item Item 
<table class="table table-striped"> ... </table>

Bordered table

Add .table-border to the class of your table.
# Col1 Col2 Col3
Row1 Item Item Item
Row2 Item Item Item 
<table class="table table-border"> ... </table>

Additionally you can add the colum border adding .table-border-colum.

# Col1 Col2 Col3
Row1 Item Item Item
Row2 Item Item Item 
<table class="table table-border-column"> ... </table>
