Responsive grid sizes
You can customize the grid behavior across different devices and screen sizes. The following column classes are available:

All sizes
You can combine different column classes to specify how the grid will be displayed on different devices. For example, if you create a grid with two columns and add the class siimple-grid-col--6 and siimple-grid-col-sm--12, your grid will have two columns on screens with sizes greater than 480px (tablets and desktop devices), and one column on mobile devices.
