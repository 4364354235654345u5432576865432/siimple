Using CSS
Use siimple colors styles and variables directly in your HTML pages
Using CSS variables
You can use the CSS variables defined in the compiled version of siimple colors. You need the color name and the lightness to use the color in your CSS classes. 
.button {
    color: var(--siimple-navy);
    background-color: var(--siimple-grey-light);
    border-radius: 5px;
    padding: 10px;
}
Using CSS classes
We also provide CSS classes to change the text and the background color using the class attribute of your HTML elements. Use siimple--color-{COLOR}-{LIGHTNESS} class to change the text color, and siimple--bg-{COLOR}-{LIGHTNESS} class to change the background color. 
<div class="siimple-btn siimple--bg-aqua siimple--color-grey-extra-light">
    My button
</div>
The base color has not lightness suffix, so you should use --siimple-{COLOR} to use the base color variable, or siimple--color-{COLOR} and siimple--bg-{COLOR} to set the base color as the text and background color.
